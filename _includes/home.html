Planet API

Planet API is a 2D planet sprite generator. Our internal process procedurally generate surfaces, map them to a true sphere, to finally render a planet at 512x512 with transparent background, lighting and solar effect.

This API currently under development. During this phase, it can be accessed for free.

Get pre-generated packs

You can directly get pre-generated sets of planets sprites from Itch.io. Many of them are free !
